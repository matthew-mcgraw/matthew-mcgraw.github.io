DL RaceBot

A Proximal Policy Optimization (PPO) agent trained to autonomously race a custom Ackermann vehicle around a rectangular track in the Webots simulator — navigating checkpoints, dodging moving obstacles, and improving lap times across five iterative training phases.

Built on the deepbots framework, the agent uses a 60-ray 180° LIDAR sensor, positional state, and orientation to learn entirely from scratch via trial and error.

RL Algorithm

Proximal Policy Optimization (PPO) with separate Actor and Critic networks. Clipped surrogate objective with γ = 0.99, clip ε = 0.2.

Training Phases

5 iterative controllers
1
Controller 01 — Baseline

4 actions (stop, forward, steer L/R). Full LIDAR rays from sensor. Drive speed = 40. Establishes core lap-completion capability.

2
Controller 02 — Refinement

Identical architecture to 01 but trained from a fresh seed. Used to compare convergence consistency and agent quality across runs.

3
Controller 03 — Moving Obstacles

Moving obstacle added to the track. Obstacle position appended to observations. Separate MovingObstacleController bounces the obstacle up and down the straight.

4
Controller 04 — Multi-Speed

Action space expanded to 5 (adds half-speed and ¾-speed). Drive max = 80. Steering penalty added. Batch size raised to 64 for more stable updates.

5
Controller 05 — Better Lap Times

Continues from a pre-trained Racer_MultiSpeed_04_Faster checkpoint. Fine-tuned for minimum lap time with the multi-speed action set.

Actor & Critic Networks

class Actor(nn.Module):
    def __init__(self, n_inputs, n_outputs):
        super().__init__()
        self.fc1 = nn.Linear(n_inputs, 128)
        self.fc2 = nn.Linear(128, 128)
        self.action_head = nn.Linear(128, n_outputs)

    def forward(self, x):
        x = F.relu(self.fc1(x))
        x = F.relu(self.fc2(x))
        return F.softmax(self.action_head(x), dim=1)

class Critic(nn.Module):
    def __init__(self, n_inputs):
        super().__init__()
        self.fc1 = nn.Linear(n_inputs, 128)
        self.fc2 = nn.Linear(128, 128)
        self.state_value = nn.Linear(128, 1)

    def forward(self, x):
        x = F.relu(self.fc1(x))
        x = F.relu(self.fc2(x))
        return self.state_value(x)

PPO Update Step

# For each batch in buffer:
V = self.critic_net(state[index])
delta = Gt_index - V
advantage = delta.detach()

action_prob = self.actor_net(
    state[index]).gather(1, action[index])

# PPO clipped surrogate loss
ratio = action_prob / old_action_log_prob[index]
surr1 = ratio * advantage
surr2 = clamp(ratio,
    1 - self.clip_param,
    1 + self.clip_param) * advantage

action_loss = -torch_min(surr1, surr2).mean()
value_loss  = F.mse_loss(Gt_index, V)

Ackermann Steering Geometry

The agent outputs a single steering angle; the controller converts it to per-wheel angles satisfying Ackermann geometry so the inner and outer front wheels track correctly.

def set_steering_angle(self, angle_rad):
    trackFront = 0.254   # metres
    wheelbase  = 0.2921  # metres
    if math.fabs(angle_rad) > 1e-5:
        angle_right = math.atan(
            1. / (1./math.tan(angle_rad)
                  - trackFront / (2 * wheelbase)))
        angle_left  = math.atan(
            1. / (1./math.tan(angle_rad)
                  + trackFront / (2 * wheelbase)))
    self.steer_right_motor.setPosition(angle_right)
    self.steer_left_motor.setPosition(angle_left)
